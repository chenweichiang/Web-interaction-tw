# 個人網頁

這是江振維的個人網站專案，用於展示個人資料、學經歷、作品集以及聯絡方式。

## 版本

當前版本：1.1.2

## 內容概述

- 關於我
- 學歷與經歷
- 展覽經歷
- 聯絡方式

## 技術實現

- HTML5
- CSS3
- JavaScript
- 響應式設計，兼容桌面和移動設備

## 特色

- 動態背景 (桌面版)
- 簡潔的白色背景 (手機版)
- 清晰的導航結構

## 文件結構

```
├── index.html          # 主頁面
├── css/
│   └── style.css       # 樣式表
├── js/
│   ├── main.js         # 主要 JavaScript 和背景動畫
│   └── audio.js        # 音訊處理模組
└── images/             # 圖片資源目錄
```
