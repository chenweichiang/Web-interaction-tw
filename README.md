# 江振維個人網站專案

這是江振維教授的個人網站專案，展示其學術成就、教學經驗以及創作展覽等內容。

## 專案概述

- **名稱**：江振維 (Chiang, Chenwei) 個人網站
- **日期**：2025年4月
- **語言**：繁體中文 / 英文
- **類型**：響應式個人學術網站

## 網站功能與特色

1. **響應式設計**：適應不同裝置，特別針對手機版有額外優化
2. **互動背景**：電腦版有互動性背景動畫，基於Canvas實現曲線粒子效果
3. **簡潔導航**：清晰的分區導航，包含關於、教育與經驗、展覽、邀請演講與工作坊、出版物以及聯絡方式
4. **雙語內容**：提供中文和英文內容，便於國際交流
5. **高效能設計**：自動偵測裝置類型，在行動裝置上停用重度動畫，確保跨平台流暢體驗

## 技術實現

- **前端**：HTML5, CSS3, JavaScript (無框架純原生實現)
- **動畫**：Canvas基於JavaScript的互動動畫，使用優化的渲染管道和空間哈希演算法提升效能
- **響應式設計**：Media queries配合JavaScript動態調整，確保在所有裝置上最佳顯示
- **性能優化**：包含記憶體管理、事件節流、物件池重用、批次渲染和空間索引等進階技術

### 技術實現細節

#### 背景動畫系統設計

背景動畫使用Canvas實現，核心組件包括：

1. **曲線系統**
   - 使用貝塞爾曲線創建平滑路徑
   - 動態生成的控制點使曲線產生自然運動
   - 滾動頁面時曲線可以隨之變化

2. **粒子系統**
   - 沿曲線路徑移動的立方體粒子
   - 使用空間哈希演算法優化粒子間互動計算
   - 當粒子接近時，自動形成連線

3. **性能優化技術**
   - 使用物件池重用Javascript物件以減少GC壓力
   - 三角函數值預計算避免高頻率計算
   - 使用空間索引降低碰撞檢測複雜度
   - 批次繪製減少Canvas狀態切換
   - 事件節流減少處理頻率

4. **智能設備檢測**
   - 自動偵測行動裝置並停用複雜動畫
   - 使用漸進式增強原則確保基本體驗
   - 適應不同解析度屏幕和DPI設定

5. **調試與性能監控**
   - 內建幀率監控系統
   - 錯誤恢復機制避免動畫崩潰
   - 開發者工具支援動態調整參數

## 使用說明

### 環境需求

- 現代網頁瀏覽器 (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
- JavaScript啟用
- 對於完整動畫體驗建議使用桌面設備

### 本地開發

1. 克隆存儲庫到本地
2. 使用任意網頁編輯器開啟`index.html`
3. 使用本地伺服器運行以測試所有功能 (推薦使用 Live Server VSCode 插件)

### 自定義與調整

- 修改 `config.js` 自定義背景動畫參數
- 調整 `css/style.css` 修改視覺風格與佈局
- 編輯 `index.html` 更新內容與結構

### 性能調整

若在低性能設備上出現卡頓：
1. 減少 `config.js` 中的粒子數量
2. 降低更新頻率與連線距離
3. 考慮完全禁用背景動畫
- 如需調整互動功能，請修改`js/main.js`和`js/audio.js`
